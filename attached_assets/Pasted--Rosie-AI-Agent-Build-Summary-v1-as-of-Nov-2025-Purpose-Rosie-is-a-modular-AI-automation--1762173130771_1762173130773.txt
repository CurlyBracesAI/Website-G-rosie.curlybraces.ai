📌 Rosie AI Agent — Build Summary (v1, as of Nov 2025)
🚀 Purpose

Rosie is a modular AI automation engine designed to power multiple business workflows (e.g. CRM email drafting, triage summaries, inventory classification) via a centralized API — with a consistent persona and structured output.

Rosie is being transitioned out of Make.com and into Flask (Python) so inputs/outputs are handled with a stable API layer, allowing flexible integration with Make, n8n, or future client apps.

🔧 Current Architecture

Hosted on: Replit
Backend: Flask (Python)
AI: OpenAI API (model TBD — currently using basic placeholder)
Frontend: Basic browser chat UI at / for interactive testing
API Endpoint(s):

POST /rosie-test — accepts JSON input and returns AI-generated output

Secrets stored in Replit:

OPENAI_API_KEY (environment variable)

🧠 Persona (System Prompt Summary)

Rosie is:

Efficient, dry, mildly sarcastic by default

Precision-first (hates vague or redundant input)

Emotionally intelligent: switches to full empathy mode for sensitive topics

Designed to always produce structured output (JSON, HTML, field mappings, etc.)

Built for CRM-style workflows, data extraction, drafting, and classification

📡 Intended Use Cases (Upcoming Endpoints)

These will map to future POST routes, e.g.:

Endpoint	Purpose
/rosie-email	Draft partner or client emails from CRM data
/rosie-triage	Analyze deal notes/activities and generate summary
/rosie-inventory	Classify scraped building contacts (owner vs sublessee)
/rosie-images	Take S3 image URLs → produce alt/tooltip text via Rekognition + OpenAI

All use cases expect:

Inputs: structured JSON from Make/n8n or internal systems

Output: structured JSON for downstream mapping back into CRM or UI

🔁 External Integration Flow (Example)
[Pipedrive] → [Make.com / n8n] → [Rosie API] → [JSON Reply] → [Make.com / n8n] → [Update Pipedrive] or [Send Email]


Make/n8n handles:

Authentication to CRM or APIs

Scheduling

Data retrieval/storage

Triggering Rosie

Rosie handles:

Reasoning, synthesis, writing

Structuring output

Enforcing personality and tone
